Usability Checklist






Summary

Overall, the VA's homepage is very aligned with best practices for accessibility. Given their audience of veterans, it seems likely that they've designed with accessiblity in mind. One image is missing alternative text, but it is simply a visual arrow, whereas all large and descriptive images do contain alternate text. There are a few redundant links, but the link text is different, and it seems valid if people have different questions to potentially direct them to the same linked pages. The webpage also has coronavirus up top as an alert, making it easy to see pertinent information quickly, especially content that may not be permanent, but is currently relevant. This homepage is very disability friendly (as it should be), but note that the text heavy page is also somewhat flat and information heavy.
