Project: Task Analysis

Task Diagram 1

Place a Custom Order


  1. Navigate to Website
  2. Browse available products
    1. Confirm quality of products
    2. Identify similar products
    3. Confirm idea for request
  3. Look for information on site
    1. See if accept custom order
    2. Look for pricing guides
    3. Identify turnaround times
  4. Click contact
  5. Collect information
    1. Name
    2. Custom product details
    3. Expected date/delivery
    4. Contact information
  6. Send inquiry

Task Diagram 2

Buy a Box of Chocolates

  1. Navigate to Website
  2. Shop available products
    1. Browse existing products
      1. Scroll through existing products
      2. Click into product details
    1. Search for specific product
      1. Search via search bar
      2. Click into product details
  3. Add product to cart
  4. Checkout
    1. Review cart
    2. Add shipping information
    3. Add payment information
    4. Review order
    5. Place order
    6. Receive confirmation
